# Formal Path Diminishing Returns Decision

Decision: pause autonomous formal source-object mining as the primary active discovery path.

Formal should not continue unchanged. It should continue only in two constrained roles:

1. Human-curated claim intake for externally supplied sharp formal challenges.
2. Benchmark/evaluation suite for Sovryn gates, because it is now good at killing weak formal candidates without fake Fund state.

## Why

- The last ten formal/source-object loops produced 0 InsightCandidates and 0 DiscoveryCandidates.
- Each new layer improved rejection specificity but not positive yield.
- The current bottleneck is not missing checks, evidence refs, replay plumbing, or Fund Gate wiring.
- The formal engine still lacks externally supplied claims with a witness/refutation that is both nonstandard and plausibly present before execution.

## Decision Table

| Option | Decision | Reason |
| --- | --- | --- |
| Continue unchanged | reject | Repeated zero-yield despite stronger gates. |
| Continue only with human-curated claims | accept as secondary path | Human curation can supply problem value and known-prior context the autonomous harvester lacks. |
| Pause | accept for autonomous mining | Stop spending cycles on generic formal object/claim mining. |
| Pivot to another domain | accept for active discovery | Empirical benchmark/data paths offer measurable deltas, richer holdouts, and less all-or-nothing proof pressure. |
| Become benchmark/evaluation suite | accept | Formal path is useful as a negative-control harness for gates and no-fake-Fund behavior. |
