# Reality-Grade Autonomy Trial

The Reality-Grade Autonomy Trial runs a bounded external-source-backed research
loop:

1. ingest at least 30 external source cards
2. create dataset cards for at least five dataset or benchmark sources
3. update knowledge inputs
4. run safe benchmark tasks across multiple domains
5. record baselines, candidate results, ablations, sensitivity checks, and
   failed runs
6. run independent reproduction attempts
7. run adversarial falsification attempts
8. update scientific knowledge artifacts
9. publish only curated public-safe output

The trial is meant to prove operational maturity, not universal scientific
validity. Every public claim remains source-bound, evidence-bound, and
limitation-bound.
