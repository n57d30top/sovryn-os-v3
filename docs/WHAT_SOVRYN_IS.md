# What Sovryn Is

Sovryn OS is a local-first evidence system for autonomous open-source research
and Open Invention workflows.

It can:

- discover and rank research opportunities,
- run Factory Mode,
- generate source cards and claim/feature matrices,
- build small prototypes and tests,
- execute validation through Node Alpha worker profiles,
- evaluate quality and falsification evidence,
- package curated public evidence,
- publish eligible artifacts into the existing public corpus repo through
  automated gates.

The core rule is:

> Agents act. Sovryn verifies. Git isolates. Policy gates. Evidence persists.

Sovryn favors bounded, reproducible, source-linked research artifacts over
unconstrained agent output.
