# What Sovryn Is Not

Sovryn OS is not:

- a patent filing system,
- a patentability opinion,
- a legal novelty opinion,
- a freedom-to-operate opinion,
- a chemical synthesis assistant,
- an exploit-development system,
- a blind LLM wrapper,
- an autopublish-anything bot.

Corpus publication can be autonomous only for the existing public corpus repo
and only after automated policy gates pass. Real standalone GitHub publication
remains governed by publication policy and explicit operator intent.

Public beta outputs still require human interpretation before use.
